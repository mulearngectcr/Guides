# What is Open Library?

Open Library is a free digital library project maintained by the Internet Archive.

It provides APIs that allow developers to search millions of books programmatically.

Official Documentation:

https://openlibrary.org/developers/api

---

# Why Are We Using Open Library?

For this project, we need learning resources.

Books are valuable because they:

* Provide structured learning material
* Cover concepts in depth
* Help validate information from videos
* Give the LLM better context when generating roadmaps

---

# Understanding API Requests

When you search for:

```text
Machine Learning
```

in Open Library, your browser sends a request similar to:

```text
https://openlibrary.org/search.json?q=Machine+Learning
```

Let's break this down:

| Part                    | Meaning         |
| ----------------------- | --------------- |
| https://openlibrary.org | Base URL        |
| /search.json            | Search endpoint |
| q=Machine+Learning      | Search query    |

The server processes the request and returns book data in JSON format.

# What Does Status Code 200 Mean?

HTTP status codes indicate whether a request succeeded.

Common codes:

| Code | Meaning      |
| ---- | ------------ |
| 200  | Success      |
| 404  | Not Found    |
| 401  | Unauthorized |
| 500  | Server Error |

A status code of 200 means the API successfully processed our request.

# Handling Missing Data

Not every book contains every field.

This can cause errors.

Problem:

```python
book["author_name"]
```

If the field doesn't exist:

```text
KeyError
```

Safer approach:

```python
book.get("author_name")
```

Even safer:

```python
book.get("author_name", ["Unknown"])
```

This means:

```text
If author_name exists:
    Return it

Otherwise:
    Return ["Unknown"]
```

# Building a Reusable Function

Instead of repeating code, create a function.

```python
import requests

def fetch_books(topic):

    url = f"https://openlibrary.org/search.json?q={topic}"

    response = requests.get(url)

    data = response.json()

    books = []

    for book in data["docs"][:5]:

        books.append({
            "title": book.get("title"),
            "author": book.get(
                "author_name",
                ["Unknown"]
            )[0]
        })

    return books
```

Usage:

```python
books = fetch_books("machine learning")

print(books)
```

---
