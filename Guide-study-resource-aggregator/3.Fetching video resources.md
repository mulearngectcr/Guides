# What Is API Authentication?

In Part 2, Open Library was completely public.

YouTube is different.

Google wants to know:

* Who is using the API
* How many requests are being made
* Whether usage limits are exceeded

Therefore, every request must include an API key.

Think of an API key like an ID card.

```text
Your App
    ↓
API Key
    ↓
YouTube API
```

Without the key, YouTube rejects the request.

# Understanding the Search Endpoint

YouTube provides:

```text
https://www.googleapis.com/youtube/v3/search
```

This endpoint allows us to search videos.

Example parameters:

```text
part=snippet
q=machine learning
type=video
maxResults=5
key=YOUR_API_KEY
```

---

# Making Your First Request

```python
import requests

url = "https://www.googleapis.com/youtube/v3/search"

params = {
    "part": "snippet",
    "q": "machine learning",
    "type": "video",
    "maxResults": 5,
    "key": API_KEY
}

response = requests.get(
    url,
    params=params
)

print(response.status_code)
```

Expected output:

```text
200
```

---

# Understanding Request Parameters

## part

```python
"part": "snippet"
```

Tells YouTube which information we want.

Snippet contains:

* Title
* Description
* Channel Name
* Thumbnails

---

## q

```python
"q": "machine learning"
```

The search query.

---

## type

```python
"type": "video"
```

Restricts results to videos only.

Without this, results may include:

* Playlists
* Channels
* Videos

---

## maxResults

```python
"maxResults": 5
```

Number of results to return.

---

# Building the Video URL

YouTube only returns:

```python
video["id"]["videoId"]
```

Example:

```text
abc123
```

To build the full URL:

```python
url = (
    "https://youtube.com/watch?v="
    + video["id"]["videoId"]
)
```

Result:

```text
https://youtube.com/watch?v=abc123
```

---

# Creating a Reusable Function

```python
import requests

def fetch_youtube_videos(topic):

    url = (
        "https://www.googleapis.com/"
        "youtube/v3/search"
    )

    params = {
        "part": "snippet",
        "q": topic,
        "type": "video",
        "maxResults": 5,
        "key": API_KEY
    }

    response = requests.get(
        url,
        params=params
    )

    videos = []

    for item in response.json()["items"]:

        videos.append({
            "title":
                item["snippet"]["title"],
            "channel":
                item["snippet"]["channelTitle"],
            "url":
                "https://youtube.com/watch?v="
                + item["id"]["videoId"]
        })

    return videos
```

# Common Errors

## Invalid API Key

Error:

```text
403 Forbidden
```

Usually means:

* Wrong API key
* API not enabled
* Quota exceeded

---

## Missing Environment Variable

Error:

```text
None
```

Check:

```python
print(API_KEY)
```

If it prints:

```text
None
```

your `.env` file isn't loading correctly.
