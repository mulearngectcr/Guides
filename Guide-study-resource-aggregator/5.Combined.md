# Complete Program Flow

Once all functions are implemented, your main program becomes surprisingly small.

```python
topic = input("Enter topic: ")
days = input("Enter number of days: ")

books = fetch_books(topic)

videos = fetch_youtube_videos(topic)

prompt = build_prompt(
    topic,
    days,
    videos,
    books
)

roadmap = generate_roadmap(prompt)

print(roadmap)
```

Notice how each function has a single responsibility:

| Function | Responsibility |
|-----------|---------------|
| `fetch_books()` | Get books from Open Library |
| `fetch_youtube_videos()` | Get videos from YouTube |
| `build_prompt()` | Create a structured prompt |
| `generate_roadmap()` | Generate roadmap using AI |

This makes the code easier to read and maintain.
